# ☀️ Energia Solar no Agronegócio: Inovação e Sustentabilidade
**Projeto desenvolvido para o Programa Agrinho 2026**

Este projeto consiste em uma página web informativa que destaca o impacto positivo da energia fotovoltaica no campo, abordando economia, redução de danos ambientais e o aumento da eficiência produtiva para o produtor rural.

## 🌾 Sobre o Tema: Placas Solares no Agro
O uso de energia solar no agronegócio brasileiro deixou de ser uma tendência para se tornar uma necessidade estratégica. Os principais benefícios abordados no projeto são:

1. **Redução de Custos:** Economia de até 95% na conta de luz, vital para sistemas de irrigação e granjas.
2. **Sustentabilidade:** Produção de energia limpa e renovável, diminuindo a pegada de carbono da propriedade.
3. **Autonomia:** Independência energética em áreas remotas onde a rede elétrica tradicional é instável ou inexistente.
4. **Valorização:** Imóveis rurais com sistemas solares instalados possuem maior valor de mercado.

## 🛠️ Tecnologias Utilizadas
A página foi construída utilizando as tecnologias fundamentais do desenvolvimento web moderno:

- **HTML5:** Estruturação semântica de todo o conteúdo, garantindo acessibilidade e uma boa leitura para os motores de busca.
- **CSS3:** Estilização visual, utilizando conceitos de **Flexbox/Grid** para garantir que o site seja responsivo (funcione bem em celulares, tablets e computadores).
- **JavaScript (JS):** Adição de interatividade.
